Dashboard de BI en Tableau

Este proyecto consiste en la creación de una dashboard de Business Intelligence (BI) utilizando Tableau, diseñada para analizar y visualizar un set de datos ficticio sobre préstamos y crédito. La dashboard está estructurada para ofrecer una visión detallada y dinámica de los datos, permitiendo a los usuarios interactuar y filtrar información clave con facilidad.


Componentes clave de la dashboard:

Barra de Datos Descriptivos

La parte superior de la dashboard presenta una barra con datos descriptivos esenciales, proporcionando una visión rápida de las métricas clave. Datos como: el valor acumulado por todos los préstamos; el valor máximo y mínimo de nuestros préstamos; el promedio de valor de los prestamos filtrados y el promedio de todos los préstamos; el total de comercios al que se ha dado préstamos y el total de prestamos devueltos.

Mapa Interactivo

Un mapa interactivo que destaca la distribución geográfica de los préstamos. Los países se pueden seleccionar directamente en el mapa, y esta selección actúa como un filtro, actualizando automáticamente todos los demás componentes de la dashboard para reflejar los datos específicos del país seleccionado.

Grádico de Áreas Acumuladas

Un gráfico de áreas muestra el valor acumulado de los préstamos o crédito en cada país a lo largo del tiempo. Este gráfico permite identificar tendencias y patrones de crecimiento en diferentes regiones. Los usuarios pueden optar por cambiar este gráfico a una vista de líneas de tendencia para una perspectiva alternativa.

Representación de Préstamos Individuales

Cada préstamo individual está representado como un punto, con el color de cada punto indicando el valor del préstamo con respecto al promedio. Este componente visualiza la distribución de los valores de los préstamos, facilitando la identificación de préstamos con valores inusualmente altos o bajos.

Barra de Filtros y Controles

Una barra de filtros y controles proporciona a los usuarios herramientas para personalizar su vista de la dashboard usando los siguientes filtros:

  • Selección de pais.
  • Rango de fechas para los datos mostrados.
  • Valor mínimo del préstamo.
  • Un filtro que funciona como opción para seleccionar que tipo de gráfico representará los valores acumulados de préstamos por pais.

Esta dashboard es interactiva dentro de la página web. Puedes jugar con ella y probar su funcionalidad.
