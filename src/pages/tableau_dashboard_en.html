BI Dashbord Built With Tableau

This project involves the creation of a Business Intelligence (BI) dashboard using Tableau, designed to analyze and visualize a fictitious dataset on loans and credit. The dashboard is structured to offer a detailed and dynamic view of the data, allowing users to interact and filter key information with ease.


Key Components of the Dashboard:

Descriptive Data Bar

The top part of the dashboard presents a bar with essential descriptive data, providing a quick view of key metrics. Data such as: the total value of all loans; the maximum and minimum values of our loans; the average value of filtered loans and the average of all loans; the total number of businesses that have received loans and the total number of loans repaid.

Interactive Map

An interactive map that highlights the geographical distribution of loans. Countries can be selected directly on the map, and this selection acts as a filter, automatically updating all other components of the dashboard to reflect data specific to the selected country.

Accumulated Area Chart

An area chart shows the accumulated value of loans or credit in each country over time. This chart allows for the identification of trends and growth patterns in different regions. Users can choose to switch this chart to a trend line view for an alternative perspective.

Representation of Individual Loans

Each individual loan is represented as a point, with the color of each point indicating the loan's value relative to the average. This component visualizes the distribution of loan values, making it easy to identify loans with unusually high or low values.

Filter and Control Bar

A filter and control bar provides users with tools to customize their view of the dashboard using the following filters:

  • Country selection.
  • Date range for the displayed data.
  • Minimum loan value.
  • A filter that works as an option to select which type of chart will represent the accumulated loan values by country.

This dashboard is interactive within the web page. You can play with it and test its functionality. Keep in mind this dashboard was built for a course done in Spanish, so the data is entirely in Spanish. The functionality remains the same though!
